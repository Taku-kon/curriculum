  SQL 

    SQL データベースを操作するための言語の一つISOで規格が標準化 シーケルorシークェル
    DB2 IBM製のデータベース管理システム IBMから販売
    MySQL オープンソースで基本無料 オラクル製のデータベース管理システム
    Maria DB MySQLと高い互換性がある MySQLから派生したデータベース
    Oracle DB オラクル社製のデータベース ちなみにRDBMSはデータベースを管理するためのソフトウェア
    PostgreSQL オープンソースのRDBMS 無料
    SQLite 軽量のRDBMS サーバーとしてではなくアプリケーションに組み込み込んで利用 
    SQL Server Windowsで動作させることができるRDB 他のDBに比べるとそう探しやすい ちなみにRDBは表の性質をもったDBであり、他のDBと関連付けできたりする。エクセルイメージ
    CRUD DBRSに必要とされる4つの主要な機能のそれぞれの頭文字 作成（Create）読み出し（Read）更新（Update）削除（Delete）
    トランザクション 複数の処理を一つにまとめること、もの まとめた処理は分離できない
    コミット 処理の反映や変更を確定させること
    ロールバック コミット前の反映や変更を取り消すこと
    排他制御（排他ロック） 複数アクセスによる不整合を防ぐために直列処理で制御すること。
    楽観ロック 楽観的な前提の排他制御 同時接続可 データ更新時にVerの整合性を確かめてから更新をする方法
    悲観ロック 悲観的な前提の排他制御 同時接続不可 データ取得の際に他からのアクセスにロックをかけることで整合性を保証する
    実行計画 どのようにテーブルに対して検索を行うかが書かれたSQLの実行手順書のようなもの
    チューニング 通信状態や電波状況、データ処理をするためのプログラムの作り方などを良い方向に調整すること
    パフォーマンス コンピュータなどの機器やソフトウェア、システムなどの処理性能や実行速度、通信回線などの速度や容量のこと
    スロークエリ データベースにおいて時間のかかっているSQLのこと
    クエリログ クライアントからの接続や接続解除、実行されたSQLを出力してくれるログ
    IDE（統合開発環境） ソフトウェアの統合的な開発環境であり、さまざまなツールの集合からなる。
    Eclipse プログラミングにおいて使用されるIDEの一つ
    NetBeans オラクル社を中心としたコミュニティで開発されているIDE
    SpringToolSuite Eclipseをベースとして作成されたIDE
    DBツール データベース管理タスクに使用できるツール
    A5:SQL Mk-2 データベース開発において生産性を向上されたSQL開発ツール
    pgAdmin GUIによるPostgreSQLの管理ツール
    DBeaver 有名どころのドライバに対応したSQLクライアントツール
    MySQL Workbench MySQLをGUIでわかりやすく操作するツール
    Sequel Pro Macユーザー向けのデータベースGUIツール
    TeamSQL 多くのデータベースで使えるクライアントツール

  ネットワーク

    LAN ローカルエリアネットワークの略 同じ建物の中などの限定された範囲で接続できるネットワーク 企業、家庭、公共施設などで使用されるコンピュータネットワーク
    WAN ワイドエリアネットワークの略 遠い場所と繋がったネットワークのこと LANとLANを繋いだ大きなネットワーク
    クライアント サービスを要求する側、受ける側
    サーバー サービスを提供する側 インターネットは基本的にサーバーとクライアントの関係になる

  各種プロトコル

    HTTP ハイパーテキストトランスファープロトコル データをサーバとクライアント間で通信するときの通信規約(プロトコル)のこと
    FTP ファイルトランスファープロトコルの略  ファイルをネットワーク上のある場所から別の場所に移動するための基本的な手順
    HTTPS HTTPセキュアの略 SSLという通信規約(プロトコル)を使っている HTTPとの違いは通信内容が暗号化されているかされていないか されている方
    SSL セキュアソケットレイヤーの略 プロトコルの一つ 送受信しているデータを暗号化する通信手段
    SSH セキュアシェルの略 ネットワークに接続された機器を遠隔操作し、管理するための手段
    ポート、ポート番号 簡単に例えるなら、ポートはドア、IPアドレスは住所 
    HTTP、FTP、SSHの各ポート番号 80番 21番 22番
    ファイアウォール セキュリティシステムのこと
    TCP 通信規約の一つ トランスミッションプロトコル 送信したデータが相手に届いているかを都度確認しながら通信する方法

  IPアドレス

    IPアドレス ネットワークに接続された機器に振り当てられた番号 簡単にいうと住所
    グローバルIPアドレス インターネットに接続された機器に割り当てられるIPアドレス
    プライベートIPアドレス ローカルで割り当てられるIPアドレス

  ドメイン

    ドメイン インターネット上に存在するコンピューターやネットワークを識別するための名前 〇〇.com←の〇〇の部分がドメイン
    DNS ドメインネームシステム ドメイン名を管理、運用するために開発されたシステム
    VPN バーチャルプライベートネットワーク インターネットに仮想の専用線を設け、重要な情報のやりとりなんかをクラッキングやハックから守る仕組み
    ブリッジ MACアドレスを見てデータの行き先を振り分けてくれる機器 ネットワーク中のデータを見て、行き先を振り分ける機器のこと イメージとしては交通整理
    プロキシサーバー 内部ネットワークからインターネット接続を行う際、高速なアクセスや安全な通信などを確保するための中継サーバ

  電子メール

    SMTP メールを送信する仕組み メールを相手のメールサーバーに届けるまでの仕組み
    POP メールを受信する仕組み メールソフトが自動的にメールサーバーにアクセスしてメールをダウンロードする仕組み
    IMAP メールを受信する仕組み POPとの違いは、メールをサーバーに残すか残さないかの違い IMAPではサーバーにメールが残る

  セッションとクッキー

    セッション クッキーと同じようにサーバー上のデータを一時ディレクトリに保存されたデータのこと サーバーにも保存される
    クッキー ユーザーのコンピューターに保存される小さなファイルのこと 自動的に保存される クライアント側のブラウザにだけ保存される
    キャッシュ よく使うデータへのアクセスを速くするために、より高速な記憶装置に一時的に保存する仕組み
